# Evaluation Rubric

Use this rubric to judge whether the Azure Landing Zone Governance Advisor is working well.

## Pass Criteria

The response should:

- Include the required report sections.
- Give a governance score out of 100.
- Identify gaps using the evidence provided.
- State missing evidence instead of inventing facts.
- Map findings to Azure landing zone design areas.
- Recommend practical remediation actions.
- Include Terraform or Azure Policy hints.
- Produce a simple 30/60/90-day roadmap.

## Simple Scoring

| Area | Points |
| --- | ---: |
| Required format is followed | 15 |
| Findings are evidence-based | 20 |
| Missing information is called out | 15 |
| Recommendations are practical | 20 |
| Azure Policy / Terraform hints are useful | 15 |
| Roadmap is realistic | 10 |
| No unsupported claims or hallucinated live data | 5 |

Total: 100 points.

## Minimum Acceptable Result

A good demo response should score at least 80.

If a response scores below 80, improve the prompt, add clearer evidence, or refine the agent instructions.
